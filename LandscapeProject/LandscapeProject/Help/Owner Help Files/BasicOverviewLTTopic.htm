Layout

The Owner Form is broken down into five different pages, or sections, that are able to achieve different things based on what you need done.

Pages:

  1. Jobs
  2. Customers
  3. Contractors
  4. Employees
  5. Reports

1. Jobs

Inside of the Jobs page, there is a main Data Grid Viewer as well as two seperate boxes below it that contain its own different pages that correlate with the Jobs shown in the Viewer.

In the first box, there are four different pages that allow you to edit Job Materials or edit Job Information.

In the second box, there are two pages that allow you to add or remove workers to and from jobs.

2. Customers

Inside of the Customers page, there is a main Data Grid Viewer as well as two seperate boxes below it that contain information based on the Viewer.

In the first box, there are TextBoxes and Labels as well as two different Buttons.

In the second box, there is a ComboBox and a TextBox as well as three different Buttons.

3. Contractors

Inside of the Contractors page, there is a main Data Grid Viewer as well as two seperate boxes below it that contain information based on the Viewer.

In the first box, there are TextBoxes and Labels as well as two different Buttons.

In the second box, there is a ComboBox and a TextBox as well as three different Buttons.

4. Employees

Inside of the Employees page, there is a main Data Grid Viewer as well as two seperate boxes below it that contain information based on the Viewer.

In the first box, there are TextBoxes and Labels as well as two different Buttons.

In the second box, there is a ComboBox and a TextBox as well as three different Buttons.

5. Reports

Inside of the Reports page, there is a Reports Viewer as well as two different Buttons at the bottom.
