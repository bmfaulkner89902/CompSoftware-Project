Data Grid Viewer

This page will go in-depth to the functionality of the Viewer on the Customers Page.

- Functionality -

The Viewer on this page will display everything that is on file for customers from a database. The Viewer will contain 7 columns that each hold different data. These columns in order are: CustomerID, FirstName, LastName, Address, Email, City and ZipCode. Every column can be edited in the Edit Customer Information page of the Second Box, except for the CustomerID column, which can never be changed.
