This page will go in-depth to the functionality of everything in the First Box.

- Functionality -

Add A New Contractor

In the First Box, Add A New Contractor, there will be seven TextBoxes that will allow you to enter the necessary information needed to add a new contractor to the database. Once you have your information entered, there will be two buttons below the TextBoxes. If you click "Add Contractor", it will add your information to the database and create that contractor. If you click "Cancel Adding Contractor", it will clear all the information that you had entered.
