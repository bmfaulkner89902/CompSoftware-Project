Second Box With Worker Pages

This page will go in-depth to the functionality of everything on each page in the Seond Box..

- Functionality -

Remove Worker From Job

On the first page, Remove Worker From Job, there will be two ListBoxes that contain any workers associated with the selected job. In the first ListBox, it will show any contractors with the job. In the second ListBox it will show any employees with the job. Below the ListBoxes there are two buttons. The first button will allow you to remove the selected contractor from the selected job. The second button will allow you to remove the selected employee from the selected job. You can not select more than one worker from inside the same ListBox at a time.

Add Worker To Job

On the second page, Add Worker To Job, there will be two ComboBoxes that contain all workers. In these ComoBoxes, you can type to search or select the drop-down arrow and choose the worker from the list. The first ComboBox will contain the names of all contractors that are in the database. The second ComboBox will contain the names of all the employees that are in the database. Below the ComboBoxes will be two buttons. Once you have selected a contractor, employee, or both, you can click the "Add Selected Contractor and/or Employee", which will add the workers to the job that is selected. If you click "Cancel Adding Workers" it will clear your selections from the ComboBoxes.
