- Functionality -

Edit Customer Information

In the second box, Edit Customer Information, there will be a ComboBox that you can use to type or select a category (which is a column from the Viewer), and proceed to the TextBox below it to enter new information. When new information has been entered, click the "Submit Changes" button to update the database with the new information. If you click "Cancel Changes", it will clear your selection and entered information. If you click "Delete Selected Customer", you will remove that customer from the database and anything associated with them.
