Data Grid Viewer

This page will go in-depth to the functionality of the Viewer on the Contractors Page.

- Functionality -

The Viewer on this page will display everything that is on file for contractors from a database. The Viewer will contain 8 columns that each hold different data. These columns in order are: ContractorID, FirstName, LastName, Email, Phone, Address, City and ZipCode. Every column can be edited in the Edit Contractor Information page of the Second Box, except for the ContractorID column, which can never be changed.
